Chromium / Chrome / Edge

Baixe o arquivo .crx ou .zip. Para testes, normalmente use chrome://extensions, ative o modo desenvolvedor e carregue a pasta descompactada.

Auto-update externo usa update_url no manifest.json e arquivo XML HTTPS.

Firefox

Baixe o .xpi. Para auto-update fora da AMO, a extensão precisa apontar para um manifesto JSON HTTPS no campo browser_specific_settings.gecko.update_url.

O XPI final precisa estar assinado para uso normal no Firefox.

Atualizações

Quando lançar uma nova versão, suba o arquivo na aba Releases do GitHub, edite data/extensions.json e o workflow recria os manifests de update.

Feito para GitHub Pages + GitHub Releases. Sem backend, sem Railway e sem custo fixo.

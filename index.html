Vivaldi / Chromium

Baixe o Vivaldi ZIP, extraia e carregue a pasta da extensão em vivaldi://extensions com o modo desenvolvedor ativado.

Extensão carregada por pasta é melhor para teste. Auto-update real no Chromium exige pacote CRX assinado e update_url.

Zen Browser / Firefox

Baixe o Zen XPI. Para teste, use a página de debug de extensões do Firefox/Zen e carregue o arquivo temporariamente.

Para instalação permanente e auto-update real, o XPI precisa estar assinado e apontar para o manifesto JSON HTTPS.

Atualizações

Quando lançar uma nova versão, troque os arquivos em downloads/, edite data/extensions.json e o workflow recria os manifests de update.

Feito para GitHub Pages + GitHub Releases. Sem backend, sem Railway e sem custo fixo.

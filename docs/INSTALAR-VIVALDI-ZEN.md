# Como instalar no Vivaldi e no Zen Browser

## Vivaldi

1. Baixe `downloads/akai-avatar-routine-vivaldi-2.1.2.zip`.
2. Extraia o ZIP.
3. Abra `vivaldi://extensions`.
4. Ative **Developer mode / Modo desenvolvedor**.
5. Clique em **Load unpacked / Carregar sem compactação**.
6. Selecione a pasta extraída da extensão.

## Zen Browser

1. Baixe `downloads/akai-avatar-routine-zen-firefox-2.1.2.xpi`.
2. Para teste, abra `about:debugging#/runtime/this-firefox`.
3. Clique em **Load Temporary Add-on**.
4. Selecione o arquivo `.xpi` ou o `manifest.json` da pasta Firefox.

## Atualizações

- Vivaldi com extensão carregada por pasta: atualização manual, baixando a nova versão e clicando em **Update** ou recarregando a extensão.
- Zen/Firefox: para auto-update permanente, o XPI precisa estar assinado e usar este update URL: `https://akaimxntis.github.io/ADeals/updates/firefox/akai-avatar-routine.json`.
